## Deployment

type "make" to compile the code

```bash
  make
```

type "./Lab3 [input] [output]" will output the output.txt that is euler path

```bash
  ./Lab3 [input] [output]
```

type "make clean" to delete .o file and binary file

```bash
  make clean
```
